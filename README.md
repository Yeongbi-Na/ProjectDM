
## DataMining Project (2021-1)
데이터마이닝입문 프로젝터 "휴게소 만족도 요인 분석"

👋 My role: 데이터 수집, 모델 개발 등
 </br>
 
## Purpose
매년 교통공사에서는 휴게소 만족도 조사 실시

휴게소 관리 입장에서는 만족도 향상에 여향을 미치는 요인을 알기 어려움

→ 만족도에 영향을 미치는 요인을 분석해 휴게소 만족도 향상 도모
